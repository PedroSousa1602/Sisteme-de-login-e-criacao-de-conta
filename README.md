# Página de Criação de Conta e Login

Este projeto apresenta duas páginas: uma para criar uma conta e outra para fazer login. Ele foi desenvolvido apenas com HTML e CSS e pode ser usado como base visual para projetos maiores.

## O que o projeto inclui

* Página de criação de conta
* Página de login
* Layout simples e responsivo
* Estilos em um arquivo CSS separado
* Estrutura fácil de adaptar para um back-end no futuro

## Estrutura do projeto

```
│── Login.html        (tela de login ou página inicial)
│── Register.html     (tela de criação de conta)
│── form.css          (estilo visual)
└── images/back.png   (imagem usada para background)
```
## Observações

* O projeto não realiza autenticação real, já que não usa JavaScript ou back-end.
* Se quiser torná-lo funcional, pode integrar com qualquer tecnologia de back-end no futuro.
